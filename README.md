# OpenHIS 1.0
### 一.系统简介

大宅门云诊所系统基于thinkphp3.2.3开发的免费开源系统，是以完整的基层医疗机构信息化解决方案为出发点，打造链接诊所、医生、患者、一站式互联网医疗服务系统，深度挖掘了基层医疗机构需求，解决其真正痛点，提供医疗前沿资源及信息共享等、全面提升医疗管理质量，可执行落地的综合性解决方案。
